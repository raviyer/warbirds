Create Interesting DTFs

Points of interest

enter field, dar and way points by clicking on the map some place. Give the POI a name, I suggest you name fields like f28, dars as dar28 and waypoints as wp1 etc. If you want a circle around one of the points simply check the box next to it in the table below, give it a radius in miles and select the color you would like

NameX posY posCircle?Radius in milesColor

Lines and Curves

enter start and end points for the line or curve, you can either enter x,y coords or names of points of interest above (enter the name in the startX and endX boxes and leave the Y values blank), Curves also require two controling points, the curve will not pass the points but go towards the first and arrive at the end point from the direction of the second control point. If you don't want to curve the line leave the values for the control points as -1. If you wanted to use just 1 controlling point then use the same values for CP1 and 2

Start XStart YEnd XEnd Y Control Point 1 XControl Point 1 Y Control Point 2 XControl Point 2 Y Avg. HUD SpeedROC ft/mt.Color

Text

enter top left coordinates, and the text you want to display and color of text. Each line is drawn by itself below the previous line

Top Left XTop Left YTextColor
